# 🐍 CodeAlpha Python Programming Internship Projects

This repository contains Python projects developed during my **CodeAlpha Python Programming Internship**. The projects focus on Python programming fundamentals, problem-solving, GUI development, file handling, and logical implementation.

## 📌 Projects Included

### 1. 🎮 Hangman Challenge

A console-based Hangman game where players guess hidden words from different categories before running out of attempts.

#### Features

* Multiple difficulty levels
* Category-based word selection
* ASCII Hangman visualization
* Score tracking system
* Performance statistics
* Replay functionality

---

### 2. 📈 Stock Portfolio Tracker

A GUI-based stock portfolio management application built using Tkinter.

#### Features

* Stock selection and quantity management
* Automatic investment calculation
* Portfolio holdings table
* Export portfolio as TXT and CSV
* Remove and clear portfolio options
* Professional dark-themed interface

---

### 3. 🤖 SmartBot – Rule-Based Chatbot

A Tkinter-based chatbot that responds to predefined user commands using rule-based logic.

#### Features

* Interactive chat interface
* Greeting and help commands
* Date and time responses
* Chat clearing functionality
* Timestamped conversations
* Dark-themed GUI

---
